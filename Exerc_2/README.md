# Exercise 2: Centered Star Triangle

## Description
This exercise displays a centered triangle made of stars (asterisks).

## What it does
The program prompts the user to enter the height of the triangle and then prints a centered triangle pattern using asterisks. Each row contains an odd number of stars (1, 3, 5, 7, etc.), centered with spaces.

## Input
- Height: The number of rows in the triangle

## Output
- A centered triangle pattern made of asterisks

## Example
```
Give the height of the triangle : 5
    *
   ***
  *****
 *******
*********
```

## File
- `CenteredStarTriangle.java`: Main program that generates the centered triangle
