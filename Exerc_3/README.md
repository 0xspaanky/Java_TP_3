# Exercise 3: Odd Squares

## Description
This exercise calculates and displays the squares of the first N odd numbers.

## What it does
The program:
1. Prompts the user to enter how many odd numbers to process
2. Stores the squares of the first N odd numbers in an array
3. Displays each odd number along with its square

## Input
- N: How many odd numbers to calculate squares for

## Output
- A list showing each odd number and its square value

## Example
```
How many values: 5
1 has square 1
3 has square 9
5 has square 25
7 has square 49
9 has square 81
```

## File
- `OddSquares.java`: Main program that calculates and displays squares of odd numbers
